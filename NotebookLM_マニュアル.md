# OSR標高管理 Pro v9 - 公式マニュアル兼概要資料

本ドキュメントは、「OSR標高管理 Pro v9」の概要、各機能の操作手順、およびスマートフォンへのインストール（登録）方法をまとめた総合マニュアルです。NotebookLM等のナレッジベースに読み込ませることで、アプリの仕様や使い方に関する精度の高い質問応答が可能になります。

---

## 1. アプリ概要 (Overview)

「OSR標高管理 Pro v9」は、建設・土木現場において「施工用基準レベル標高値（O.P.）」をスマートフォン上で直感的に管理・記録するための専用ウェブアプリケーションです。
現場での紙とペンによる記録をデジタル化し、計算ミスを防ぐとともに、事務所での報告書作成業務を大幅に効率化します。

### 主な特徴
- **PWA（Progressive Web App）対応**: アプリとしてスマホのホーム画面にインストール可能。オフライン環境でもサクサク動作します。
- **現場特化型UI**: 屋外の直射日光下での視認性を高めた「ライトモード」と、屋内の暗所環境に適した「ダークモード」の切替機能。手袋をしていても押しやすい専用の「大型10キーパッド」を搭載。
- **自動計算機能**: 初期値や前月値と、今回入力した「当月値」を比較し、自動的に「変位差（変動量）」を計算してPDF帳票に記載します。
- **多彩な出力形式**: 現場で入力したデータをその場で「A4横サイズのPDF」としてプレビュー・ダウンロードできるほか、Excel等で再利用可能な「CSV形式」でのエクスポートにも対応しています。
- **データ保護**: ローカルストレージを利用した高速なデータ保存に加え、全データをJSONファイルとして一括保存・復元できる「完全バックアップ機能」を備えています。

## 2. インストール・登録方法 (Installation)

本アプリはアプリストア（App Store / Google Play）を経由せず、ウェブブラウザから直接スマートフォンにインストールします。

### 準備事項
アプリの公開URL（例： `https://taka2560-commits.github.io/hyoko-survey-app/` ）にアクセスします。

### iPhone (iOS) での登録手順
1. **Safari**ブラウザで公開URLを開きます。
2. 画面下部の中央にある **「共有アイコン（四角から上矢印が飛び出したマーク）」** をタップします。
3. 表示されたメニューを少し下にスクロールし、**「ホーム画面に追加」** を選択します。
4. 右上の **「追加」** をタップすると、ホーム画面に専用のアイコンが作成されます。

### Android での登録手順
1. **Google Chrome**ブラウザで公開URLを開きます。
2. 画面右上の **「︙（3点リーダー）」** をタップしてメニューを開きます。
3. **「ホーム画面に追加」** または **「アプリをインストール」** を選択します。
4. 確認画面で「追加」をタップすると、ホーム画面に専用のアイコンが作成されます。

※以降は、ホーム画面のアイコンをタップして起動することで、ブラウザのアドレスバーの無い「全画面アプリモード」として快適にご利用いただけます。

## 3. 操作手順 (How to Use)

### 3.1. 現場（プロジェクト）の管理
1. 画面上部の **「⚙️ 管理」** ボタンをタップします。
2. 「新規現場の追加」欄に新しい工事名（例：新港湾整備工事）を入力し、「追加」ボタンを押します。
3. 不要になった現場は、リストの左側にあるチェックボックスを選択し、「選択した現場を削除」ボタンで消去できます。
4. 画面左上のドロップダウンリストから、入力を行いたい現場を選択して切り替えます。

### 3.2. 計測データの入力
1. 画面中央のタブで **「📍 水準点鋲」** または **「🏷️ 測量シール」** を選びます。
2. リストから入力したい「点名」の行をタップします。（選択された行は色が変わりアクティブになります）
3. 画面下部の **「10キーパッド」** を使って当月の標高値（O.P.）を入力し、**「ENTER」** を押すと確定されます。
4. 前月と全く変動がない場合は、10キー内の **「前月値コピー」** ボタンを押すだけで、前月値がそのまま当月値として自動入力されます。

### 3.3. 点の追加と削除
1. タブの右横にある **「✏️ 追加・削除」** ボタン（整理モード）をタップします。
2. **新しい点の追加**: 画面下部に現れる「➕ 新しい点名を追加」ボタンを押し、点名、初期値、前月値、備考を直接テーブル上でキーボード入力します。
3. **点の削除**: 削除したい行の左端にあるチェックボックスをONにし、画面下部の「🗑️ 選択した点を一括削除する」ボタンを押します。
4. 編集が終わったら、再度「✏️ 追加・削除」ボタンをタップして通常モードに戻ります。

### 3.4. 帳票出力（PDF / CSV）
- **📄 PDF出力**: 
  画面上部の「PDF出力」ボタンを押すと、A4横レイアウトの印刷プレビュー画面が開きます。
  「📥 PDFダウンロード」ボタンで、スマホ内にPDFファイルとして保存できます。「🖨️ 通常の印刷」ボタンでWi-Fiプリンタ等への直接印刷も可能です。
- **📊 CSV出力**: 
  画面上部の「CSV出力」ボタンを押すと、現場の全データがカンマ区切りのCSVファイルとしてダウンロードされます。現場から事務所のパソコンに送信してExcel等で編集する際に利用します。

### 3.5. 月次更新機能
1. 画面右上の **「月次更新」** ボタンをタップします。
2. 確認ダイアログで「確定する」を選ぶと、現在の「当月値」が「前月値」にスライド移行し、「当月値」が空欄にリセットされます。
3. これにより、翌月の新しい計測をスムーズに開始できます。

### 3.6. データのバックアップと復元
1. 画面上部の **「⚙️ 管理」** ボタンをタップします。
2. 画面下部の **「💾 バックアップ保存」** を押すと、全現場のデータが1つのJSONファイルとしてスマホにダウンロードされます。
3. 万が一誤ってデータを消してしまった場合や、スマホを買い替えて新しい端末に移行する場合は、**「📂 ファイルから復元」** を押し、保存しておいたJSONファイルを選択することで、すべてのデータが元の状態に完全復元されます。
