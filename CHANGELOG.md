# 更新履歴 (Changelog)

## v9.2.0 (2026-06-28)
- **機能強化**: 管理表の構成に合わせ、アプリのタブを「水準点 / KBM」「シール / 1級」「3級(構台等)」の3タブ構成に変更。
- **データ更新**: 新規測点として3級基準点（アングル、山留、構台杭）を初期テンプレートに追加。
- **UI調整**: 3タブ化に伴い、PDF印刷プレビュー画面のレイアウトを3カラム出力に対応。

## v9.1.0 (2026-05-23)
- **機能追加**: CSVデータ出力機能を追加。現場データをExcel等で活用可能になりました。
- **機能追加**: 全データバックアップ（JSONエクスポート）および復元（インポート）機能を追加。「管理」画面から利用可能です。
- **機能追加**: READMEおよび更新履歴（CHANGELOG）ファイルを追加。
- **PWA対応**: manifest.json および Service Worker (sw.js) を実装し、スマホでのホーム画面追加・全画面起動に完全対応。
- **デザイン**: 専用のアプリアイコン（icon.png）を追加。

## v9.0.0
- **初期リリース**: OSR標高管理 Pro v9 として基本機能を実装。
- タブ切り替え（水準点鋲・測量シール）、専用10キーパッド、PDF印刷プレビュー機能、ダークモード切替機能を搭載。
