# OSR標高管理 Pro v9 (PWA)

施工用基準レベル標高値(O.P.)を現場のスマートフォンで簡単に管理・入力するためのウェブアプリケーションです。
ブラウザ上で動作し、PWA（Progressive Web App）としてホーム画面に追加することで、ネイティブアプリのように全画面で利用可能です。

## 主な機能
- **測点の階層管理**: 1級基準点(測量シール)、2級基準点(水準点・KBM)、3級基準点(構台・アングル等)の3タブ構成で管理
- **水準点・測量シールの管理**: 初期値、前月値、当月値の記録と変位差の自動計算
- **テンキー入力**: スマホでの屋外作業に最適化された専用テンキーパッド
- **PWA対応（オフライン対応）**: 現場の電波が悪い場所でもサクサク動作
- **PDF帳票出力**: 現場で入力したデータをその場でA4横サイズのPDF（3カラム出力）として出力可能
- **CSVデータ出力**: Excelなどでの二次利用のためにデータをCSVエクスポート
- **バックアップ・復元**: アプリ内のデータをJSONファイルとして一括保存し、機種変更時などに簡単に引き継ぎ可能
- **ダークモード対応**: 屋内の暗い環境でも見やすいテーマ切り替え

## 使い方（インストール方法）
1. スマホのブラウザ（iOSはSafari、AndroidはChrome推奨）で公開URLにアクセスします。
2. ブラウザのメニューから「ホーム画面に追加」をタップします。
3. ホーム画面に追加されたアイコンからアプリを起動して使用します。

## 開発・環境
- HTML / CSS / Vanilla JavaScript
- Tailwind CSS (CDN)
- html2pdf.js (クライアントサイドPDF生成)
